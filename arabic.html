*-----( أنواع الواو )-----*

{ واو الحال }

تدخل الواو الحالية على الجملة الاسمية والجملة الفعلية شرط أن يسبقها أحد أنواع المعارف مثل: “زارني محمد وهو سعيد” هنا الواو حالية لأن محمد اسم معرفة.

{ واو المعية }

هي الواو التي بمعنى (مع) تدخل على الجملة ومن مواضع واو المعية إذا تلاها اسم يعرب مفعولاً معه وإذا تلاها فعل يعرب فعل مضارع منصوب مثل: “سرت والبحر” أو “أكتبُ واجباتي وأخرجَ للعبِ”.

{ واو العطف }

هي الواو التي تعطف جملة على جملة أو تعطف اسماً مفرداً على اسم مفرد مثل: “سامر حاضر وخليل غائب” ومثل “حضر سامر وخليل” وهذا الفرق بين واو المعية وواو العطف.

{ واو الإشباع }

هي الواو التي تقع بعد ضمير الرفع المتصل (تاء الرفع المتحركة) أو بعد ضمير النصب المتصل (كاف الخطاب) مثال: “هل أديتم الواجب وفهمتموه؟” ورد في قوله تعالى “فأسقيناكموه”.

{ الواو الزائدة }

هي الأحرف التي تزاد على الميزان الصرفي للأفعال المجردة فتصبح مزيدة بحرف أو بحرفين أو بثلاثة أحرف وهي مجموعة بكلمة “سألتمونيها”

{ واو الاستئناف (الواو الابتدائية) }

هي الواو التي تقع بعدها جملة لا علاقة لها بالجملة التي قبلها وتعرب الجملة التي بعدها استئنافية لا محل لها من الإعراب مثال: قال تعالى ﴿ثم قضى أجلاً وأحل مسمى عنده).

{ واو رُبّ }

هي الواو التي تأتي بمعنى (رُبَّ) وهي حرف من حروف الجر للاسم النكرة مثال: ” وليلٍ كموجِ البحرِ أرخى سدولَه “.

{ الواو التي حسب ما قبلها }

هي الواقعة في بداية البيت الشعري إذا أتى وحده مثال: ” وزائـرتـي گــأن بـهــا حـيـــاءٌ*** فـلـيــــس تـــزور إلا فــي الـــظلامِ “.

{ واو الجماعة }

هي ضمير رفع منفصل يتصل بالأفعال للدلالة على أن الفاعل جمعاً أي أكثر من اثنين مثال: ” سينتصرون في الحرب” أو ” الأطفال لعبوا بالكرة”.

{ واو الأسماء الخمسة }

وهي واو علامة رفع الأسماء الخمسة أي عندما يقع أحد الأسماء الخمسة مرفوع وهي (أب أخ حم ذو فو) مثال: “قدم أبوك” أو محمدٌ ذو جاه”.

{ واو القسم }

هي حرف جر دالة على لفظ الجلالة والقسم مثال: قال تعالى ﴿وَالسَّماء والطارق) أو (والشام درةُ الحضارات) أي أنني أقول والله الشام درة الحضارات.

{ واو الجزاء }

هي الواو التي تنصب الفعل المضارع بـ (أن) المضمرة بعد واو الجزاء جوازاً مثال: “طالع الكتب وتستفيدَ خير لك من اللهو”.

{ واو الفصل }

هي الواو المحصورة في بعض الكلمات وهذه الكلمات: (عمرو، أولاء، أولئك، أولي، أولو، أولات(.

{ واو جمع المذكر السالم }

هي الواو علامة رفع جمع المذكر السالم مثال: “حضر المعلمون” أو “المعلمون نشيطون” أو “معلمو الروضة ماهرون”

{ واو الثمانية }

هي الواو التي يقع بعدها كلمة دالة على العدد ثمانية مثال: قال تعالى ﴿حتى إذا جاءوها وفتحت أبوابها﴾ ويعتقد النحويون أن واو الثمانية غير حقيقية ولم يعترفوا بها وليس لها قواعد إعراب.

( الفرق بين الواو في الإعراب )

{ واو الحالية وواو العطف }

واو العطف تفيد في عطف اسمين على بعضهما قاما بنفس الفعل ولهما نفس حركة الإعراب أو عطف فعلين على بعضهما قام بهما نفس الفاعل.

{ واو المعية وواو العطف }

واو العطف تفيد في عطف اسمين على بعضهما قاما بنفس الفعل ولهما نفس حركة الإعراب أو عطف فعلين على بعضهما قام بهما نفس الفاعل.
مثال): رسمَ محمد لوحة وعلقَّها) هنا اشترك الفعلان وهما الرسم والتعليق بسامر الذي قام بالفعلين معاً إذاً الواو هنا حرف عطف.

{ أما واو المعية }

لا تُفيد اشتراك ما قبلها وما بعدها في الحكم، بل تدل على المُصاحبة ، ويعرب الاسم الذي يليها مفعول معه. مثال: مشيت على البحر والعصرَ. (كلمة العصرَ هنا مفعول معه، ولا يصح أن تكون معطوفة على البحر لأن المعنى للجملة أنني مشيت على البحر في العصر.
لكي تكون الواو معية يجب أن يكون هناك مانع للعطف مثال: “سأمت منك وربيعاً” هذه الواو ليست حرف عطف حتماً فلا يجوز عطف اسم على جار ومجرور أو فعل أي أن معنى الجملة سأمت من الاثنين معاً وممكن تحويل هذه الجملة لعطف كالتالي: سأمت منك ومن ربيع.
تكون الواو للعطف إذا كانت تفيد في الترتيب والتعقيب مثال: “ذهبَ محمد وعلي إلى المدرسة” كما يجوز إعرابها مفعول معه على النحو التالي “ذهب محمد وعلياً إلى المدرسة” والمقصود هنا أن محمد مع علي ذهبوا للمدرسة.[1]

{ واو الإشباع وواو الزائدة }

واو الإشباع يسبقها ضميرين تاء الرفع وكاف الخطاب مثال: ساعدتموه في البذار؟ أما الواو الزائدة تزاد في الميزان الصفي للفعل مثال: “صبور على وزن فعول مزيدة بالواو وأصلها فعل”.

{ واو رب وواو القسم }

واو رب هو حرف جر يقدر تقديراً مثال: “وليل كموج البحر” أي ورب الليل كموج البحر أما واو القسم حرف جر أيضاً وهو نوع من أنواع القسم مثال: ” والشمس وضحاها” أي يقسم الله بالشمس.

{ واو الجماعة واو جمع المذكر السالم وواو الأسماء الخمسة }

واو الجماعة هو الضمير المتصل بالأفعال يعرب في محل رفع فاعل ويتصل بالأفعال الماضية والمضارعة والأمر مثال: ألعبوا/ يلعبون/ لعبوا أما واو جمع المذكر السالم وهي الواو التي تأتي علامة إعراب جمع المذكر المرفوع مثال: “حضر المعلمون” وهنا كلمة المعلمون فاعل مرفوع بالواو لأنه جمع مذكر أو “المعلمون نشيطون” أو “معلمو الروضة ماهرون” وهنا معلمو مبتدأ مرفوع بالواو وحذفت النون للإضافة أما واو الأسماء الخمسة فهو علامة رفع الأسماء الخمسة مثال: “رأى أبوكَ صديقك” وهنا كلمة أبوك فاعل مرفوع بالواو لأنه من الأسماء الخمسة.

{ واو الجزاء وواو الفصل }

هي الواو التي تنصب الفعل المضارع بـ (أن) المضمرة بعد واو الجزاء جوازاً مثال: “طالع الكتب وتستفيدَ خير لك من اللهو” ويعرب الفعل مضارع فعل مضارع منصوب بأن المضمرة بعد واو الجزاء، أما واو الفصل هي الواو المحصورة في بعض الكلمات وهذه الكلمات: (عمرو، أولاء، أولئك، أولي، أولو، أولات(.

{ واو الاستئناف }

هي الواو التي تقع بعدها جملة لا علاقة لها بالجملة التي قبلها وتعرب الجملة التي بعدها استئنافية لا محل لها من الإعراب مثال: قال تعالى ﴿ثم قضى أجلاً وأحل مسمى عنده) مثال: ثم راحَ يعزف ويغني محمد يغني أي أن غناء محمد لا دخل له بعزفه فكل منهم يقوم بحدث منفرد
